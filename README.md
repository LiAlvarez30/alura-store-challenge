# Alura Store — Challenge Data Science (ONE)

## Objetivo
Analizar el desempeño de 4 tiendas de Alura Store para recomendar cuál debería vender el Sr. Juan, considerando:
- Facturación total
- Categorías más vendidas
- Calificación promedio de clientes
- Productos más y menos vendidos
- Costo promedio de envío

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab

## Estructura del proyecto
- `AluraStoreLatam.ipynb`: notebook con el análisis completo y visualizaciones
- `base-de-datos-challenge1-latam/`: archivos CSV por tienda (del repo base)

## Principales hallazgos
- La tienda con menor facturación total es **Tienda 4**.
- Las categorías líderes se concentran principalmente en **Muebles** y **Electrónicos**.
- La mejor calificación promedio es de **Tienda 3**.
- El envío promedio más bajo es de **Tienda 4**.

## Recomendación final
Se recomienda vender **Tienda 4**, ya que presenta la **menor facturación total** y no compensa esa diferencia con una ventaja suficientemente fuerte en otras métricas.

## Cómo ejecutar
1. Abrir `AluraStoreLatam.ipynb` en Google Colab o Jupyter Notebook.
2. Ejecutar las celdas en orden (no modificar la sección de carga de datos).
